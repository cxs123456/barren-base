
# barren-base

## 项目简介
后台管理系统

## 工程结构

```bash

barren-base
└── barren-core                       # 系统核心模块
     ├── barren-common                # 公共模块，常用工具集
     ├── barren-starter-auth          # 认证授权
     ├── barren-starter-web           # web相关
     └── barren-starter-log           # 日志
└── barren-system                     # 系统启动模块
├── barren-tools                      # 系统第三方工具模块
└── barren-generator                  # 代码生成模块

```
